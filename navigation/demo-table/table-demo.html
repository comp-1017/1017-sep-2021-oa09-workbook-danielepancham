Table Investigation

A table is a way of creating a structured grid display of data. The tags we will be interested in are:

  • <table> : identifies the table
  • <thead> : identifies the header row of the table
  • <th> : identifies the text in the head of the column
  • <tbody> : identifies the body of the table (detail rows)
  • <tr> ; identifies the start of a row
  • <td> : identifies the a data cell on the body row
  • <tfoot> : identifies the footer row of the table

All tags are paired tags (opening and closing tag). The border parameter on the table tag will allow you to specify a border around your table. The colspan parameter and rowspan parameters are used on the <td> tab to allow you to merger columns on a single row and colums cells across rows.
